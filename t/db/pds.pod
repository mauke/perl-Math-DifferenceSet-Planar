=head1 NAME

pds.db - small database with planar difference set samples

=head1 VERSION

This is version 0.9 of the database, created on August 29, 2019.

=head1 DESCRIPTION

This file is a small sample database only used for testing.

It contains 7 cyclic planar difference sets with orders ranging from
2 to 9.  The sets are chosen minimal by lexicographic ordering.

=head1 FILE FORMAT

SQLite 3.x database.

=head1 SCHEMA

  CREATE TABLE difference_set (
    order_   integer NOT NULL PRIMARY KEY,
    base     integer NOT NULL,
    exponent integer NOT NULL,
    modulus  integer NOT NULL,
    n_planes integer NOT NULL,
    deltas   text    NOT NULL
  );
